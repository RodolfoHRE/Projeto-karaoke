        Bem-vindo ao echo karaokê 🎤

        Pra buscar músicas por nome você precisa de uma chave gratuita da YouTube Data API. Leva ~2 min. Ou pule e use colando links/IDs do YouTube.

        1. Abra o Google Cloud no botão abaixo e crie um projeto.
        2. Habilite a YouTube Data API v3 (a página já abre nela).
        3. Vá em Credenciais → Criar credenciais → Chave de API.
        4. Copie a chave e cole aqui embaixo.
